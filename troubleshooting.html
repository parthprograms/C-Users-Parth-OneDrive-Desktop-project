Problem Solution
Battery not charging Check charger connection, ensure power source is working, and verify battery health in the app. If issue persists, contact service center.
Throttle unresponsive Check for error codes on display, restart the scooter, and inspect throttle wiring for damage.
Reduced range Check tire pressure, avoid frequent hard acceleration, and ensure brakes aren't dragging. Update to latest firmware.
Connectivity issues Restart the mobile app, ensure Bluetooth is enabled, and check for app updates in your app store.
Problem Solution
Scooter shuts down randomly Check battery connectors and ensure they're secure. Update controller software at nearest service center.
LED display flickers Inspect fuse box and secure wiring connections. Check for moisture in display unit.
Charging port issues Clean charging port with dry cloth, ensure charger is properly connected. Avoid using damaged charging cables.
Unusual motor noise Check for debris in motor area. If noise persists, visit service center for motor inspection.
Problem Solution
Loose handlebar Tighten handlebar stem screws with appropriate torque. Do not overtighten.
Brake squealing Clean brake pads with isopropyl alcohol or replace if worn out. Check brake alignment.
Software glitches Perform a system reset by holding power button for 10 seconds. Visit service center for updates.
Range anxiety Use Eco mode for longer range, plan routes with charging stations, and avoid excessive weight.
Problem Solution
Scooter vibrates on uneven roads Check shock absorbers for leaks and tire pressure. Ensure all bolts are properly tightened.
App connectivity issues Reinstall the TVS Connect app, ensure location permissions are granted, and check Bluetooth connectivity.
Charging time increased Check power source voltage, use original charger, and avoid charging in extreme temperatures.
Error codes on display Note down error code and refer to manual. Common codes can be cleared with a system restart.
